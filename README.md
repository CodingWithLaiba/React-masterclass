# React-masterclass

# ⚛️ React Masterclass – Code Stoic

A complete hands-on repository following the **ReactJS Masterclass: Beginner To Advanced** series by [Code Stoic](https://www.youtube.com/playlist?list=PLSsAz5wf2lkKm0BG9wUWWSgYWBzDa-dFs).

Each folder contains the code from the corresponding video/topic.

---

## 📁 Project Structure

| Folder                    | Topics Covered                                  |
| ------------------------- | ----------------------------------------------- |
| `1-creating-components`   | JSX, Functional Components, Component Structure |
| `2-Props`                 | Passing Props, Props Destructuring              |
| `3_RendringArray`         | Rendering Lists, Array Methods, Keys            |
| `4-Conditional-component` | Conditional Rendering, Ternary, Short-circuit   |
| `5-Events`                | Event Handling, onClick, onChange               |
| `6-States`                | useState Hook, Forms, Controlled Components     |
| `7-UseReducer`            | useReducer Hook, State Management               |
| `8-ReactRouter`           | React Router v6, Pages, Navigation, 404         |
| `9-ContextAPI`            | Context API, useContext, Global State           |
| `10-CustomeHooks`         | Custom Hooks, localStorage Hook                 |
| `TodoApp`                 | Full Project – CRUD Todo App                    |
| `FoodRecipeApp`           | Full Project – API Calls, useEffect, Search     |

---

## 🚀 How to Run Any Project

```bash
# Go into any folder
cd FoodRecipeApp

# Install dependencies
npm install

# Start dev server
npm run dev
```
